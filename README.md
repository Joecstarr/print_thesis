# Thesis for print

This LaTeX document builds the text and cover for self publication by [blurb](https://www.blurb.com).

The documents are configured for a $6\times9$ trade paperback. At time of
writing it costs about \$10 per copy to buy from Blurb.

## Building 

To build the document ensure you have [tectonic](https://tectonic-typesetting.github.io/en-US/) and [just](https://github.com/casey/just) installed. Then run:

```sh
just go
```
